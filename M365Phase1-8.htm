How do I access Outlook on the Web?

One way to do this is to use a Google Chrome or Edge browser to go to office.com. Enter your Summa email and password, go through the required security steps. Then once there, choose Outlook from the menu.

If you want to go directly to Outlook on the Web, go to outlook.office.com.

How do I fix my time zone in Outlook on the Web? Its wrong.

To correct the time zone in Microsoft 365 Outlook on the Web, select the Settings gear icon in the top right corner then click on "View all Outlook Settings" at the bottom of that menu. Select General in the settings box and change time zone on the right side.

Why is everything so slow in Microsoft 365?

We recommend using Microsoft 365 Outlook on the Web through Google Chrome or Edge. This has helped with speed.

Where are my desktop notifications while using Outlook on the Web?

You can only guarantee desktop notifications if using Microsoft 365 Outlook on the Web in Google Chrome. This is done within Outlook on the Web by going to the settings gear icon in the top right corner. And there is a Desktop notification toggle to turn on right there in that menu.

Do I have to go through Okta and Duo to access anything in my Microsoft 365 account?

You will be automatically routed through Okta when you sign into any Microsoft 365 apps on the web. However, you will only be prompted for Duo if you are not on a Summa Health device or network. Either way, once these security steps are initially set up, they are fast and easy to get through.
